What is this project about?

This is a visual help to see the aggregated effects of different Flexbox settings.

HOVER

TO LEARN MORE ABOUT IT:

  • visit website
  • check thed code

Is this a niche project? Not at all! There are other available tools exist on the Internet to achive the same task. The reason I created this app so I can get my head around the differences among align-items, align-content, align-self settings and other similarly confusing titles, also to visualize and experiment the aggregated effects of multiple Flexbox settings.

The code behind the project

I created it with vanilla JavaScript. The JS code is based on plenty of Event Listeners, separated into two modules.

The backbone of the CSS layout -although this project is about Flexbox- is CSS grid. There is no special reason for that apart from my personal preference.

Have fun, and play around with this app!
